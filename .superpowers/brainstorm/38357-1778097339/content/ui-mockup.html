Single monitor item

Bar shows live key values inline. Clicking expands a popover with all metrics + sparklines.

- type: monitor
  name: Mac Studio
  adapter: dashdot
  url: http://192.168.1.10:3001
  interval: 10

Group of monitors

A named group in the bar opens a dropdown listing all machines and their live stats.

- type: group
  name: Servers
  items:
    - type: monitor
      name: Mac Studio
      adapter: dashdot
      url: http://192.168.1.10:3001
    - type: monitor
      name: NAS
      adapter: glances
      url: http://192.168.1.20:61208
